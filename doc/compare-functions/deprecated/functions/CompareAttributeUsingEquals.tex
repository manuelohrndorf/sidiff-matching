\comparefunction{CompareAttributeUsingEquals}

\allowedtypes
numerical Values, eString, eDate, eChar, eCharObject, eBoolean, eBooleanObject,
Enumeration, datatype, eByteArray

\precondition
-

\semantics
Compares two nodes based on the string values of their attributes. 

\returnvalue
In case the values of the attributes are equal sim = 1 is returned, else sim =
0.

\parameters
For eStrings: The name of the attributes and the case-sensitive value
For other attributes: Only the attribute to be compared by name\\


\exceptions
